# 📊 Customer Churn Analysis

This project analyzes customer churn behavior using telecommunication data.

## 🔍 Methods Used
- Descriptive Statistics
- Data Visualization
- Logistic Regression

## 📌 Key Insights
- Contract type strongly influences churn
- Customers with higher monthly charges are more likely to churn
- Longer tenure reduces churn probability
